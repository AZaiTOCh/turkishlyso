## Mathematical / formal (tokenish coinages)

| Term | Definition |
|------|------------|
| **IVV operator** (*I viv*) | **I**nverse **V**ice **V**ersa — single turnkey on an ordered pair: reverse **and** component-wise negate. Example: IVV(2, 35) = (−35, −2). Inverse alone: e.g. toward (−2, 35) in the founder’s illustration of sign-flip on the first slot’s story; vice versa alone: (35, 2). **IVV** does both in one step → (−35, −2). |
| **IVV symbol** | **`<>`** — ASCII glyph (avoids ⟡ / U+27E1, already common on calculators). Written `(a, b) <> ↦ (−b, −a)`. |

### IVV — prior art check (honest)

| Nearby math/CS idea | Same as IVV? |
|---------------------|--------------|
| Contrapositive (¬q → ¬p) | Related *spirit* (swap + negate) for **logic**, not ordered numeric pairs |
| (f ∘ g)⁻¹ = g⁻¹ ∘ f⁻¹ | Order reverse + inverses for **functions** |
| Reflection through the line *y = −x* | **Same map** on ℝ²: (a,b) ↦ (−b,−a) — standard geometry; **not** previously branded “IVV” |
| Named “IVV / Inverse Vice Versa” operator | **Not found** as a standard named operator in common math/CS references |

**Verdict:** The **map** is classic linear geometry; the **IVV name + turnkey pairing of inverse∐vice-versa** is a tokenish pedagogical/formal coinage. Suitable for the glossary; not a claim of a new Euclidean isometry.
