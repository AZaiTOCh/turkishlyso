# Resgents

**Class:** Engineship value under the [Agentic Registry](AGENTIC_REGISTRY.md)  
**Incepted:** v0.5.0 (TOKISH induction)

## Definition

**Resgents** are native agentics with Engineship = **Resgent**: grown from or specialized for the mother codebase — lighter/tighter than general **Agent** Engineship (plug-in / migrant patterns).

**Analogy:** LPUs vs GPUs — specialized lean units vs general external/plugged-in agents.

## Engineship vs Status

| | |
|--|--|
| **Engineship** | Agent **or** Resgent — “citizenship” of the agentic relative to an engine |
| **Status** | Plug-in · Reserve · Resident — how it sits in the tree *now* |

See registry columns and evolution rules in [AGENTIC_REGISTRY.md](AGENTIC_REGISTRY.md).
